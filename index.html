1. Pengenalan Variabel

Memahami konsep dasar variabel dalam JavaScript

Apa itu Variabel?

Variabel adalah tempat penyimpanan data yang dapat berubah-ubah. Dalam JavaScript, kita bisa mendeklarasikan variabel dengan tiga cara:

  • var - cara lama (function scope)
  • let - cara modern (block scope)
  • const - untuk nilai yang tidak berubah

4. Operator JavaScript

Mengenal berbagai jenis operator untuk manipulasi data

Jenis-jenis Operator

JavaScript memiliki berbagai operator untuk melakukan operasi pada data:

// Contoh dasar operator let x = 10; let y = 3; // Aritmatika console.log(x + y); // 13 console.log(x - y); // 7 console.log(x * y); // 30 console.log(x / y); // 3.333... console.log(x % y); // 1 (sisa bagi) // Perbandingan console.log(x > y); // true console.log(x == y); // false console.log(x === y); // false (strict equality) // Logika let isTrue = true; let isFalse = false; console.log(isTrue && isFalse); // false console.log(isTrue || isFalse); // true console.log(!isTrue); // false

Aritmatika Math

Operasi matematika dasar

+, -, *, /, %, **

Perbandingan Compare

Membandingkan nilai

==, ===, !=, !==, >, <, >=, <=

Logika Logic

Operasi boolean

&&, ||, !

Assignment Assign

Memberikan nilai

=, +=, -=, *=, /=

5. Kondisi (If-Else)

Membuat keputusan dalam program dengan kondisi

Statement Kondisi

Kondisi memungkinkan program membuat keputusan berdasarkan situasi tertentu. JavaScript menyediakan beberapa cara:

  • if - eksekusi jika kondisi benar
  • else if - kondisi alternatif
  • else - eksekusi jika semua kondisi salah
// Struktur dasar if-else let umur = 18; if (umur >= 18) { console.log("Dewasa"); } else { console.log("Belum dewasa"); } // Multiple conditions let nilai = 85; if (nilai >= 90) { console.log("A"); } else if (nilai >= 80) { console.log("B"); } else if (nilai >= 70) { console.log("C"); } else { console.log("D"); } // Ternary operator (kondisi singkat) let status = umur >= 18 ? "dewasa" : "anak-anak"; console.log(status);

6. Perulangan (Loops)

Mengulang kode dengan efisien menggunakan loop

Jenis-jenis Loop

Loop memungkinkan kita mengulang kode tanpa menulis berulang-ulang:

For Loop Count

Loop dengan jumlah iterasi yang diketahui

for (let i = 0; i < 5; i++)

While Loop Condition

Loop selama kondisi benar

while (condition)

Do-While Execute First

Eksekusi dulu, baru cek kondisi

do { } while (condition)

For-Of Array

Loop untuk array/string

for (let item of array)
